# Security Policy

## Supported Versions

Only the latest `1.x.x` version of the Lynx MI MCP Server receives security updates.

| Version | Supported          |
| ------- | ------------------ |
| 1.x.x   | :white_check_mark: |
| 0.x.x   | :x:                |

## Reporting a Vulnerability

If you discover a security vulnerability within the Lynx MI MCP Server or the underlying API integration, please securely report it to us instead of using public issue trackers.

1. **Email:** Send an email to [security@lynx-mi.com](mailto:security@lynx-mi.com).
2. **Details:** Include a clear description of the vulnerability, steps to reproduce, and any known potential impact.
3. **Response:** Our security team monitors this inbox and will acknowledge receipt within 48 hours. We endeavor to resolve critical issues immediately.

We appreciate all responsible disclosures that help keep the Lynx MI platform and our user community safe.
